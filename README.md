# SIGNLEARN

SIGNLEARN adalah aplikasi pembelajaran bahasa isyarat berbasis web yang memanfaatkan Artificial Intelligence dan Computer Vision untuk membantu pengguna mempelajari BISINDO dan SIBI secara interaktif dan real-time.

## ✨ Features

* Modul pembelajaran BISINDO & SIBI
* Real-time hand gesture detection
* AI gesture recognition menggunakan CNN
* Quiz interaktif
* Riwayat belajar pengguna
* Admin dashboard

## 🧠 Tech Stack

### Frontend

### Backend

* TensorFlow
* TensorFlow.js

* MySQL

## 🚀 Installation

```bash id="i0m7kg"
# Clone repository
git clone https://github.com/username/signlearn.git

# Masuk ke folder project
cd signlearn

# Install dependencies
pip install -r requirements.txt

# Run server
uvicorn main:app --reload
```
